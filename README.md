# RoleSync
Роли на сайте (VIP в чате, доступ в разделы) можно выдавать и снимать автоматически по тому, что известно о игроке на сервере через **GiveCore**: есть ли VIP в игре, набран ли порог опыта и т.д. Правило звучит простым языком: «если выполняется набор условий — выдать роль X», причём условия можно объединять «и» внутри группы и «или» между группами,
